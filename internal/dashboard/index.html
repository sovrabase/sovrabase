🚀 Quick Start

Create your first Sovrabase project and get an API key instantly.

Add Sovrabase to your Flutter app:

final sovrabase = SovrabaseClient( apiKey: 'YOUR_API_KEY', projectId: 'YOUR_PROJECT_ID', );

Use the REST API with your project credentials:

# List collections curl -H "X-Project-Key: YOUR_API_KEY" \ -H "Authorization: Bearer USER_TOKEN" \ http://localhost:6070/api/v1/collections/users # Create a document curl -X POST \ -H "X-Project-Key: YOUR_API_KEY" \ -H "Authorization: Bearer USER_TOKEN" \ -H "Content-Type: application/json" \ -d '{"name":"Alice"}' \ http://localhost:6070/api/v1/collections/users
